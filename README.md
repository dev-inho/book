# 전국 도서관 앱

## 목적
- 전국 도서관의 위치, 운영시간 등에 관련한 기본 정보와 어떠한 도서가 있는지를 사용자에게 알려주고자 함.

## 화면
- 로그인 페이지
- 메인 페이지
- 도서관 설명 페이지
- 도서 목록
- 도서 상세
- 열람실 현황

# 모델
- User (미구현)
- Book (구현)
- Library (미구현)
